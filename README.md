Project Title: Emotion Detector - 
The purpose of this project is to develop a machine learning model and application that allows users to upload images and receive real-time emotion predictions. 
The dataset used is FER-2013, with 35k images in total.
The emotions that can be detected are Happy, Sad, Angry, Fear, Disgust, Surprise, and Neutral.
Technologies used are Jupyter Notebook, Python, Streamlit (for the app), and libraries such as Pandas, Numpy, Matplotlib and Seaborn (for visualisation).
